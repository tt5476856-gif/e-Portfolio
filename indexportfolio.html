Section 3 — Individual Contributions

I led the implementation of secure authentication (bcrypt hashing, JWT session tokens) and maintained the threat model and risk register. I coordinated threat-mitigation tasks via GitHub issues and assisted with penetration testing and remediation. Tools used included GitHub, Jira/Trello, OWASP ZAP, and local CI tests. Challenges involved dependency management and balancing feature delivery with security hardening; these were addressed through targeted research and automated vulnerability scanning.

Section 4 — Evaluation: Final Project vs Unit 6 Status

The Unit 6 status document set ambitious goals; by Unit 11 we had scaled back non-essential features to prioritise secure core functionality. The final product shows measurable improvements in authentication strength, input validation, and documented incident response procedures. Teamwork matured from ad-hoc task assignments to a clearly defined sprint-like cadence with retrospectives and role clarity.
